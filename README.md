# Nocachemode
CDNキャッシュを回避するために、URLへ毎回異なるクエリパラメータ（キャッシュバスター）を付与するChrome拡張機能です。
対象ドメインの登録、現在開いているサイトの追加、全URLモードの切り替えに対応しています。

## 主な機能
- ドメイン/URLを手動で追加
- 現在開いているタブのURLを追加
- 登録済みドメインの一覧/削除
- 全URLモード（すべてのhttp/httpsでキャッシュバスターを付与）

## 仕組み
- `http://` / `https://` の遷移のみを対象にします。
- `cacheBypassMode` が有効なら全URLへ `_upd_` を付与します。
- 無効なら登録済みドメインに一致した場合のみ `_upd_` を付与します。

## インストール
1. `chrome://extensions/` を開きます。
2. 右上の「デベロッパー モード」をオンにします。
3. 「パッケージ化されていない拡張機能を読み込む」から `bypass-cache-ext` を選択します。

## 使い方
1. 拡張機能アイコンをクリックしてポップアップを開きます。
2. 「ドメイン/URLを追加」または「現在のサイトを追加」で登録します。
3. 一覧から不要なドメインは削除できます。
4. 「全URLモード」をオンにすると、すべてのhttp/httpsでキャッシュバスターを付与します。

## 注意点
- 本拡張は `http://` / `https://` のみを対象にします。
- `_upd_` パラメータが既に付いている場合は再付与しません。
- 設定は `chrome.storage.sync` に保存されます。
